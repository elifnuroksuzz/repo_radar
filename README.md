# 🚀 Repo Radar

GitHub kullanıcı profillerini detaylı analiz eden modern web uygulaması. Repository'ler, programlama dilleri ve aktivite geçmişini görselleştirir.

## 🌟 Öne Çıkan Özellikler

### 👤 **Kullanıcı Profil Analizi**
- Detaylı GitHub profil görüntüleme
- Takipçi/takip edilen analizi
- Hesap yaşı ve aktivite durumu
- Bio ve iletişim bilgileri

### 📊 **Repository İstatistikleri**
- Toplam repository sayısı
- En popüler repository'ler (⭐ yıldız sayısına göre)
- Fork edilmiş projeler
- Son güncellenen repository'ler
- Repository büyüklük analizi

### 💻 **Programlama Dili Analizi**
- Kullanılan programlama dillerinin dağılımı
- Dil bazında kod satırı istatistikleri
- Interaktif pasta grafikleri
- Trend analizi ve karşılaştırma

### 📈 **Aktivite Görselleştirme**
- Commit geçmişi heat map'i
- Haftalık/aylık aktivite grafikleri
- Contribution streak takibi
- Zaman bazlı aktivite analizi

### 🎨 **Modern UI/UX**
- Responsive tasarım (📱 mobil uyumlu)
- Dark/Light tema desteği
- Smooth animasyonlar
- Kullanıcı dostu arayüz

## 🛠️ Teknoloji Stack'i

### Frontend
- **React.js** ⚛️ - Modern UI framework
- **JavaScript ES6+** - Modern JavaScript özellikleri
- **CSS3/SCSS** - Responsive ve modern styling
- **Chart.js** 📊 - Veri görselleştirme
- **React Router** - Single Page Application routing

### API & Backend
- **GitHub REST API v4** 🔗 - Gerçek zamanlı veri
- **Fetch API** - HTTP istekleri
- **JSON** - Veri formatı

### Araçlar & Kütüphaneler
- **Webpack** - Module bundler
- **Babel** - JavaScript transpiler
- **ESLint** - Code linting
- **Prettier** - Code formatting

## 🚀 Hızlı Başlangıç

### Ön Gereksinimler
- Node.js (v14.0.0 veya üstü)
- npm veya yarn
- Modern web tarayıcısı

### Kurulum

1. **Repository'yi klonlayın:**
```bash
git clone https://github.com/elifnuroksuzz/repo_radar.git
cd repo_radar
```

2. **Bağımlılıkları yükleyin:**
```bash
npm install
# veya
yarn install
```

3. **Geliştirme sunucusunu başlatın:**
```bash
npm start
# veya  
yarn start
```

4. **Tarayıcınızda açın:**
```
http://localhost:3000
```

## 🔧 Yapılandırma

### GitHub API Token (Opsiyonel)
Daha yüksek API limiti için GitHub Personal Access Token ekleyebilirsiniz:

1. `.env` dosyası oluşturun
2. Token'ınızı ekleyin:
```env
REACT_APP_GITHUB_TOKEN=your_personal_access_token_here
```

## 📱 Kullanım

### 1. GitHub Kullanıcısı Arama
- Ana sayfada GitHub kullanıcı adını girin
- "Analiz Et" butonuna tıklayın
- Sonuçların yüklenmesini bekleyin

### 2. Profil Keşfi
- **Overview Tab**: Genel profil bilgileri
- **Repositories Tab**: Repository listesi ve detayları  
- **Languages Tab**: Programlama dili analizi
- **Activity Tab**: Commit aktivite haritası

### 3. Veri Filtreleme
- Repository'leri yıldız sayısına göre sıralama
- Dil bazında filtreleme
- Tarih aralığı seçimi
- Arama ve filtreleme seçenekleri

## 🤝 Katkıda Bulunma

Projeye katkıda bulunmak isterseniz:

1. **Fork** yapın
2. Feature branch oluşturun:
   ```bash
   git checkout -b feature/harika-ozellik
   ```
3. Değişiklikleri commit edin:
   ```bash
   git commit -m "✨ Harika özellik eklendi"
   ```
4. Branch'i push edin:
   ```bash
   git push origin feature/harika-ozellik
   ```
5. **Pull Request** oluşturun

### 📝 Katkı Kuralları
- Code style'a uyun (ESLint + Prettier)
- Test ekleyin (mümkünse)
- Commit mesajlarında emoji kullanın
- Documentation güncelleyin
